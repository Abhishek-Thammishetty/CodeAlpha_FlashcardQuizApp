# Flashcard Quiz App
A simple Flashcard Quiz Application developed as part of the CodeAlpha App Development Internship.
## Features
- View flashcards
- Show answers
- Next and Previous navigation
- Add new flashcards
- Edit existing flashcards
- Delete flashcards
- Data stored using Local Storage
- Responsive and user-friendly interface
## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
## How to Run
1. Download the project.
2. Open index.html in a browser.
3. Start studying with flashcards.
